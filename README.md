# 📊 Data Analyst Candidate Selection Tool using Python, Pandas, and Matplotlib

---

## 📌 Project Overview

The Data Analyst Candidate Selection Tool is a Python based Data Analytics project that automates the process of reading, analyzing, filtering, and selecting candidates with Data Analysis skills from an Excel dataset.

This project simulates a real world HR Analytics workflow where raw candidate data is transformed into meaningful insights and automated reports using Python, Pandas, and Matplotlib.

It demonstrates core Data Analyst skills including Data Cleaning, Data Filtering, Data Visualization, Excel Automation, and Python based data processing.

This project reflects real world responsibilities of a Data Analyst working with structured business data.

## ⚙️ How the Project Works

### Step 1: Reads Excel Dataset  
The program loads candidate data from an Excel file into a Pandas DataFrame.

### Step 2: Cleans and Preprocesses Data  
Column names and dataset structure are cleaned to ensure accurate processing.

### Step 3: Filters Data Analyst Candidates  
Candidates with Data Analysis skills are filtered using Pandas.

### Step 4: Selects Top Candidates  
Relevant candidate information is extracted and organized.

### Step 5: Generates Automated Excel Report  
Filtered candidates are exported into a new Excel file automatically.

### Step 6: Visualizes Data  
Candidate skill distribution is visualized using Matplotlib charts.
